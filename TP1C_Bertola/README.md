Trabajo practico 1C consiste en la implementación de aplicaciones cliente-servidor en Python, utilizando sockets TCP.
Utilizamos dos versiones del servidor:
- Servidor concurrente (usando threads)
- Servidor con concurrencia aparente (usando select)
Ambas versiones permiten el envío de archivos desde el cliente al servidor.
